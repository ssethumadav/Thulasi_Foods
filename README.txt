TO DEPLOY ONLINE:
-----------------
Option A — Netlify (Free, easiest):
  1. Go to https://netlify.com → Sign up free
  2. Drag & drop the index.html file onto the dashboard
  3. You get a live URL instantly

Option B — GitHub Pages (Free):
  1. Create a GitHub account at https://github.com
  2. New repository → upload index.html as "index.html"
  3. Settings → Pages → Branch: main → Save
  4. Your site is live at https://yourusername.github.io/repo-name
